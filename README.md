# Hack Aotearoa 2020: Exploring the eICU Collaborative Research Database

Example notebooks for [Hack Aotearoa 2020](http://hackaotearoa.co.nz/).

## Introduction

The notebooks in this repository can be used to explore the [eICU Collaborative Research Database](https://eicu-crd.mit.edu/), a research database created from hundreds of unaffiliated hospitals across the USA.
